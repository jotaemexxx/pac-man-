# Pac-Man (ASP.NET Core)

Um jogo de Pac-Man desenvolvido com **ASP.NET Core** e **SignalR**

## Tecnologias
- ASP.NET Core
- C#
- SignalR (real-time)
- HTML

## Como rodar o projeto localmente

1. Clone o repositório
2. Abra a solução `PacMan.sln` no Visual Studio
3. Pressione `F5` para executar

## Estrutura do Projeto

- `src/WebApplication1/` → Código principal do jogo
- `wwwroot/` → Arquivos do frontend (HTML, JS, CSS, imagens)
